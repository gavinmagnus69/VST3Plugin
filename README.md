# VST3Plugin

**VST3Plugin** is a standalone plugin written using C++ and Juce. It provides the user with the ability to interact with the delay of one of the audio streams.

## Installation

Make sure that the following programs are installed on the computer:

- [Juce](https://juce.com/download/) - C++ framework for audio application and plug-in development.
- [Visual Studio](https://visualstudio.microsoft.com/ru/) - A standalone IDE running on Windows, macOS and Linux.
- [Git client](https://git-scm.com/) - Git is a free and open source distributed version control system.

Then
- Clone this repository:
```bash 
git clone https://github.com/gavinmagnus69/VST3Plugin.git
```
- With **Projucer** open *tc_dlr.jucer* file.
- From **Projucer** open project in **Visual Studio**. 
- Compile and run project, all executable files will be located in the builds folder

## Usage

Open settings and setup input and output channels, and other necessary customization. To start synchronization press sync button in main menu.
